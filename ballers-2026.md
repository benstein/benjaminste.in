# Oakland Ballers 2026 Home Games

A subscribable calendar of all 44 [Oakland Ballers](https://oaklandballers.com) 2026 home games at Raimondi Park. Each event has the date and first-pitch time in Pacific Time, the opponent, the promo night theme (Bay Mitzvah, Punk Night, KPOP Demon Bunters, etc.), the venue address, and a link to the ticket page.

This is unofficial — I just wanted it on my own calendar and figured I'd publish it.

## Subscribe

A subscribed calendar stays in sync with this URL, so if I update anything (corrections, additions) your calendar picks it up.

- **Apple Calendar / iPhone:** [Click to subscribe](webcal://benjaminste.in/assets/ballers-2026.ics). Calendar will open and ask you to confirm.
- **Google Calendar:** Open [Google Calendar settings](https://calendar.google.com/calendar/u/0/r/settings/addbyurl), then paste `https://benjaminste.in/assets/ballers-2026.ics` into "From URL." Google refreshes subscribed calendars roughly every 24 hours.
- **Outlook:** Add a calendar → Subscribe from web → paste the same `https://` URL.

## Download

If you'd rather just import the events once (no auto-sync), grab the file directly: [ballers-2026.ics](/assets/ballers-2026.ics).

## Source

Dates, times, opponents, and promo themes scraped from [tickets.oaklandballers.com](https://tickets.oaklandballers.com/). If a game gets rescheduled or a theme changes, the source of truth is the Ballers ticket page, not this calendar. Open an issue or email me if you spot a mismatch and I'll fix it.
